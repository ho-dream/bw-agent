# harness-bw-agent

用 harness 的方式复现 [bw-agent](../bw-agent) 的成果——构建一个全自主 AI 开发管线（黑灯工厂）。

## 核心理念

- **黑灯工厂模式**: 最小化人工干预，AI 自主完成开发任务
- **模型自迭代**: AI 在 harness 框架内自行迭代，无需人工逐步指导
- **Harness 优先**: 我们只构建基础设施，不关心 AI 具体的任务产出

## 技术栈

- [opencode](https://opencode.ai) + opencode plugins
